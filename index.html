LaravelMart - Complete eCommerce Platform

Features list supported:

FrontEnd - User area

  • Login
  • Register
  • Billing - Address, Payment type, Delivery type
  • Wishlist
  • Apply coupon
  • Cart - add, remove, edit products
  • View products
  • View product

Backend - Admin area

  • Manage Users
  • Manage Payment types
  • Manage Products
  • Manage Categories
  • Manage Coupons
